## 💡 核心价值

### 1️⃣ 实战导向，非纸上谈兵

基于 **100+ 制造企业实地调研**，每个场景描述都来自真实工位观察，而非文献推测。

### 2️⃣ 量化评估，支撑决策

**七维评估模型**为每个场景提供量化打分：
- 人工替代紧迫性 → 需求真实性
- 规模化潜力 → 复制价值
- 超人类价值 → 增量收益
- 操作复杂度 → 技术门槛
- 结构化程度 → 环境要求
- 容错空间 → 风险等级
- 节拍压力 → 时间约束

### 3️⃣ 标准化元动作，通用语言

**三级元动作体系**建立了人形机器人能力描述的标准语言，支持：
- 跨场景能力对比
- 训练数据规范制定
- 技能迁移评估

### 4️⃣ 开放共享，持续演进

遵循 **CC BY-SA 4.0** 协议开源，欢迎：
- 补充新场景
- 校准技术参数
- 分享实践经验
