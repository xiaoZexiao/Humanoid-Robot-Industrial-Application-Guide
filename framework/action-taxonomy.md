# 元动作分类体系说明

## 概述

元动作（Primitive Action）是人形机器人执行工业任务时的原子级动作单元。本项目建立了一套三级分类体系，用于标准化描述和组织这些动作。

## 层级结构

```
L1 大类（9个）—— 基于运动学本质划分
  └─ L2 中类（65个）—— 基于操作特征细分
      └─ L3 实例（122+个）—— 具体场景中的原子动作
```

## L1 大类定义

| L1编号 | 名称 | 英文 | 核心特征 | L2数量 |
|--------|------|------|----------|--------|
| L1-01 | 全身位移 | Whole-body Locomotion | 整体质心空间平移 | 8 |
| L1-02 | 躯体姿态调整 | Torso Posture Adjustment | 支撑面不变，身体构型改变 | 7 |
| L1-03 | 上肢空间运动 | Upper Limb Spatial Motion | 手臂在空间中的移动 | 8 |
| L1-04 | 手部抓握操控 | Hand Grasping & Manipulation | 手指与物体的接触控制 | 7 |
| L1-05 | 线性力施加 | Linear Force Application | 沿直线方向施加力 | 7 |
| L1-06 | 旋转操作 | Rotational Operation | 绕轴旋转的操作 | 7 |
| L1-07 | 约束路径操作 | Constrained Path Operation | 沿预定义路径的运动 | 7 |
| L1-08 | 表面随形运动 | Surface Conforming Motion | 贴合表面的运动 | 7 |
| L1-09 | 感知定位运动 | Perception-guided Motion | 基于感知反馈的运动 | 7 |

## L3 编码规则

```
L3-[L2编号]-[三位序号]

示例：L3-04-01-003 表示 L2-04-01（力量型抓握）下的第3个L3实例
```

## L3 命名规范

```
[动作动词] - [对象特征] - [关键约束]

示例：
- 抓取-袋装物料-双手对握
- 旋拧-六角螺栓-定扭矩
- 插入-连接器-精密对位
```

## L3 标准字段

| 字段 | 类型 | 说明 |
|------|------|------|
| l3_id | string | 唯一编号 |
| l3_name_zh | string | 中文名称 |
| l3_name_en | string | 英文名称 |

| description | string | 动作描述 |
| object_category | string | 操作对象类别 |
| parameter_ranges | object | 参数范围 |
